Web browsers and suitable os platforms

The listing based on the analysis of the visitors and shall show how many different Web browser on several os platforms will be used in the World Wide Web (WWW). The listing is not complete, only Web browsers used on aadmm.de are listed. Table 1 list the os platforms Linux, MacOS, Solaris and Windows, Table 2 the os platforms AmigaOS, BeOS, OS/2 and Unix, Table 3 several PDA browsers and Table 4 list a unsorted list of other browsers.
